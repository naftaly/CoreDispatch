# CoreDispatch

CoreDispatch is a simple wrapper around GCD.

## Installation

CoreDispatch is available on [CocoaPods](http://cocoapods.org). Just add the following to your project Podfile:

```ruby
pod 'CoreDispatch'
```

## Usage

Use it as follows:

```objective-c
#import <CoreDispatch/CoreDispatch.h>
 ```
